Goal of the project:

Analyzing given two playlist and finding common features such as energy,popularity and so on.
Than creating a new playlist on user account based on given playlists.
The more similar the playlists are, the better the results will be.

Requirements:

1.You have to create an app from "https://developer.spotify.com/dashboard/applications".
2.Than you have to get in the project you created and get your client_ID and client_Secret.
3.After getting those, go to "Edit Settings" and create a redirect uri such as "http://localhost:8888/callback"

How to use:

Enter your information to "spoticonfig.py" and run "recommendation.py".
Make sure that both files are in the same directory.
Enter the spotify URL to program and give permission from the pop-up authentication window.
